# TypeScript libraries

- **lodash-es** — object/type helpers: `isObject`, `isFunction`, `pick`/`omit`, `cloneDeep`, etc.
- **luxon** — date/time: `DateTime`, `Duration`, `Interval`, `Settings`
- **type-fest** — type utilities: `ValueOf`, `ReadonlyDeep`, `ArrayValues` (alias `typefest`)
- **neverthrow** — Result types: `ok`, `err`, `okAsync`, `ResultAsync`; prefer over try/catch where appropriate
