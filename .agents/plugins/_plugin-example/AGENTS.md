# Plugin example

## Purpose

Doc and example for building and maintaining AI agent plugins; excluded from all marketplaces (sync and Cursor logic skip it). Reference when generating or updating any plugin.

## Requirements

- Context lives in [rules/](rules/); use `.md` only in rules (agent-agnostic)
- When adding a plugin: follow [add-new-plugin](rules/add-new-plugin.md) and [plugin-conventions](rules/plugin-conventions.md)
- Do not add AGENTS.md in plugin sub-folders; rules speak for themselves
- Structure mirrors [Cursor starter-advanced](https://github.com/cursor/plugin-template/tree/main/plugins/starter-advanced); see [Figma mcp-server-guide](https://github.com/figma/mcp-server-guide/tree/83b11c0bc199a3c6ee87df0a0bd460139fa72cdc) for dual Cursor/Claude layout
