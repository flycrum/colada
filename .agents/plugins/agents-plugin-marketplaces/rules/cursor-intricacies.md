# Cursor intricacies

- No local plugin install from repo; Cursor uses [Cursor Marketplace](https://cursor.com/marketplace) only
- Colada: root [AGENTS.md](https://github.com/flycrum/colada/blob/main/AGENTS.md); optionally `.cursor/rules/` symlinks or thin `.mdc` to `.agents/plugins/*/rules`
- Do not add Cursor postinstall or symlink script; document only
- Docs: [Cursor Plugins](https://cursor.com/docs/plugins), [Building plugins](https://cursor.com/docs/plugins/building), [Rules](https://cursor.com/docs/context/rules), [Third-party hooks](https://cursor.com/docs/agent/third-party-hooks)
