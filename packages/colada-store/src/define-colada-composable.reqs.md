# define-colada-composable

Package overview: [colada.reqs.md](./colada.reqs.md).

## Purpose

Composable interface built on [define-colada-structure](./define-colada-structure.reqs.md). Config-driven via [define-colada-structure-accessors-config-map](./define-colada-structure-accessors-config-map.reqs.md); supports constructor/init-call props so the resulting composable can accept refs, values, callbacks when consumed. Implementation: [define-colada-composable.ts](./define-colada-composable.ts).
